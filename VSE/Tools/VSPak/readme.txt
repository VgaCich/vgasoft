Программа для создания пакетов VSE Package.
Позволяет создавать пакеты (по файлу проекта), автоматически генерировать файл проекта по заданной директории, просматривать содержимое пакета и извлекать файлы из пакета.

Использование:

Обозначения:
  <param> - обязательный параметр (вводится без скобок)
  [param] - необязательный параметр (вводится без скобок)

VSPak <file.vpp>
  Создание пакета по файлу проекта file.vpp

VSPak /l <file.vpk>
  Просмотр содержимого пакета file.vpk

VSPak /x [options] <file.vpk>
  Извлечение файлов из пакета file.vpk. Опции:
    /d<DIR> - извлечь файлы в директорию DIR. По умолчанию файлы извлекаются в текущую директорию.
    /m<MASK> - извлечение файлов по маске MASK. Если в маске задана директория, то извлекаются только файлы из заданной директории и её поддиректорий.

VSPak /c [options] <dir>
  Автоматическая генерация файла проекта по директории dir. Опции:
    /p<COMPRESSION> - установить сжатие файлов в COMPRESSION. Сжатие задается в виде comp/level, см. в описании формата файла проекта.
    /a - запросить опции сжатия у пользователя
  Если задать одновременно опции /p и /a, то сжатие определит та, которая будет последней в командной строке.

Общие опции:
  /w - ждать нажатия клавиши после завершения

Формат файла проекта:

Расширение: vpp
Комментарии начиная с // и до конца строки.
Первая строка проекта (не комментарий и не пустая) - базовая директория проекта - все пути указываются относительно нее.
Остальные строки - список файлов и директорий:

Описание директории:
<dirname>
список файлов и поддиректорий
<|>
где dirname - имя директории

Описание файла:
filename|comp/level
где:
  filename - имя файла
  comp - алгоритм сжатия (LZMA, NRV, Store(без сжатия), Discard(удаляет файл из виртуальной файловой системы PakMan'а))
  level - уровень сжатия:
    для LZMA: Fast, Normal, Max, Ultra
    для NRV: 1...10
    для Store: отсутствует
