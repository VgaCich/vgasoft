Для компиляции потребуются библиотеки:
1) AvL, частично. Качать эти файлы:
http://vgasoft.googlecode.com/svn/AvL/AvL.pas
http://vgasoft.googlecode.com/svn/AvL/avlMath.pas
http://vgasoft.googlecode.com/svn/AvL/avlUtils.pas
http://vgasoft.googlecode.com/svn/AvL/avlSyncObjs.pas
http://vgasoft.googlecode.com/svn/AvL/avlVectors.pas
2) Замены системных модулей, для Delphi 7 качать здесь:
http://vgacich.narod.ru/SysDCU-D7.7z
В настройках проекта предполагается, что замены установлены в $(DELPHI)\Lib\sysdcu
С другими заменами возможно потребуется закомментировать модуль SysSfIni в секции uses файла AssaultLite.dpr

Сборка:
1) Откройте группу проектов Tools\Tools.bpg, скомпилируйте все (Project->Build All) (для компиляции редактора моделей PMEdit потребуется установить компоненты из Tools\PMEdit\Components, для сборки Assault Lite редактор не требуется)
2) Выполните скрипт PreBuild.bat
3) Откройте проект AssaultLite.dpr и скомпилируйте его
4) Выполните скрипт PostBuild.bat
5) Для уборки мусора выполните скрипт clean.bat

Дефайны:

VSEDEBUG - сборка отладочной версии движка VSE
VSE_LOG - включить лог
SYNTEX_USELOG - включить лог в SynTex'е
MEMPAK_PLAINFS - использовать данные из папки Data (иначе из MemPak.inc)
DEBUGMEM - отладка памяти (требуется FastMM)
PM_DRAWNORMALS - рисовать нормали у моделей
VSE_ESC_EXIT - принудительный выход по escape

Рабочий набор дефайнов:
VSEDEBUG;VSE_LOG;SYNTEX_USELOG;MEMPAK_PLAINFS
