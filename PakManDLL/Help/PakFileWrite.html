function PakFileWrite(F: Cardinal; const Buffer; Count: Longint): Longint;

Записывает данные в файл.
F - идентификатор файла.
Buffer - буфер с записываемыми данными.
Count - размер буфера (количество байт, которые нужно записать)
Возвращает количество реально записанных байт.
Примечание:
Запись возможна только в обычные файлы, файлы в пакетах - только для чтения.
