Флаги

Функции PakCreateFile и PakOpenFile принимают флаги, аналогичные флагам TFileStream: pmShareCompat, pmShareExclusive, pmShareDenyWrite, pmShareDenyRead, pmShareDenyNone. Для функции PakOpenFile эти флаги имеют смысл, только если открываются обычные файлы.
Также эти функции принимают флаги:
pmNoCreate - не создавать файл, если открываемый файл не существует (имеет смысл только для PakOpenFile).
pmNoCheck - не проверять контрольную сумму файла (не рекомендуется). Не имеет смысла для обычных файлов и PakCreateFile.
pmNoDelete - не удалять существующий файл (имеет смысл только для PakCreateFile).
