function PakDeleteMountPoint(MountPoint: PChar): Boolean;

Удаляет точку монтирования.
MountPoint - точка монтирования.
Возвращает true, если точка монтирования успешно удалена. Подробнее о точках монтирования: Общие сведения.

Примечания:
  1. Точка монтирования не демонтируется. Она только исключается из списка точек монтирования, которые будут смонтированы при следущей инициализации менеджера пакетов.
  2. Список точек монтирования не затрагивается функциями PakInit и PakFree.
