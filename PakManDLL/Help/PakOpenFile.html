function PakOpenFile(Name: PChar; Flags: Cardinal): Cardinal;

Открывает файл.
Name - имя файла.
Flags - флаги. Список флагов
Возвращает идентификатор открытого файла или InvalidPakFile, если файл открыть не удалось.
Примечание:
Если файл не найден и флаг ofNoCreate не задан, то будет создан новый файл.
