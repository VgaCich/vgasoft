function PakFindFiles(Mask: PChar; Recursive: Boolean): PChar;

Ищет файлы, соответствующие заданному критерию.
Mask - маска имени файла, стандартного формата.
Recursive - искать ли файл в поддиректориях.
Возвращает строку с полными именами найденных файлов, разделенными символами CRLF (#13#10).
Примечание:
Возвращенный указатель указывает на правильные данные только до следущего вызова функции PakFindFiles.
